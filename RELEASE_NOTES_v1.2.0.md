## Bible Verse MOTD v1.2.0

Replaces the stump Message of the Day board with random Bible verses.

### Downloads

| File | What it's for |
|------|---------------|
| **BibleVerseMOTD-FullInstall-v1.2.0.zip** | **Recommended** — full setup with BepInEx + Utilla + mod |
| `BibleVerseMOTD.dll` | Mod only (if you already have BepInEx) |

### Full install (new users)

1. Download **`BibleVerseMOTD-FullInstall-v1.2.0.zip`**
2. Extract into your **Gorilla Tag folder** (where `Gorilla Tag.exe` is)
3. Launch Gorilla Tag and visit the stump

**Included in full package:**
- BepInEx 5.4.23.5
- Utilla 1.7.0
- Doorstop 4.5.0
- Bible Verse MOTD 1.2.0

### Mod-only install

1. Download `BibleVerseMOTD.dll`
2. Place in `Gorilla Tag/BepInEx/plugins/`

### Requirements

- Gorilla Tag (Steam PC)

### Features

- New verse on game launch and room join
- Replaces both text and image update boards
- 65 Bible verses included
- Client-side only (only you see it)
- Configurable via `BepInEx/config/com.frostytagyt.bibleversemotd.cfg`
